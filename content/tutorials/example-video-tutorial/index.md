## Embedding the Video

To embed a YouTube video, you have two options:

### Option 1: Use youtube_id in frontmatter
Set the `youtube_id` field in the frontmatter above to the video ID.

### Option 2: Manual embed
Or include the embed code directly in the content:

```html
<iframe width="560" height="315"
  src="https://www.youtube.com/embed/YOUR_VIDEO_ID"
  frameborder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowfullscreen>
</iframe>
```
