A friend asks: "Does creatine actually improve focus?" You dig through the papers, find the nuance, explain it — and that knowledge disappears into a chat thread.

What if your expert review fed a shared database anyone could search? 🧬

That's what we built: an AI-guided peer review system where Claude interviews the scientist step by step — structured, auditable, reusable. Not a chatbot. A protocol.

Every review goes into a common pool. The next person asking the same question finds real expert reasoning, not a hallucinated summary.

Stack: Next.js + Supabase + Anthropic Files API + GitHub for audit trails.
